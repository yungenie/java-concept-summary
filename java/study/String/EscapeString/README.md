● Escape string(이스케이프) 이란?   
\백슬래쉬를 앞에 붙여 java에 정의되어 있는 원래의 의미를 벗어나는(escape) 문자들을 이스케이프 문자라고 합니다.

● 정규 표현식 문자열 처리?  
정해진 규칙을 escape 하려면 \ 를 앞에 붙이거나, \Q ... \E 로 감싸야 한다.    
괄호  < ( { [ ] } ) >     
특수문자  ^ - = $ ! | ? * + .    

  1. []으로 감싸주면 문자자체로 인식하는 것들 : * + $ |   
  2. \\를 붙여줘야 하는 것들 : (){}^[]
