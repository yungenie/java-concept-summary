## 1. 클래스의 이해 기초
### 클래스의 개념
#### 객체
- 우리가 표현할 수 있는 모든 것들

#### 클래스(class)
- 객체와 관련된 데이터와 처리동작을 한데 모은 것
- 필드(멤버변수) + 메서드
- 클래스를 이용해 객체 생성 (Object o = New Object())
- o : 객체변수, 참조변수


#### 패키지(package)
- 모두 소문자로 구성
- 인터넷 도메인 이름의 역순으로 이름 구성 
- ex)github.com -> com.github
